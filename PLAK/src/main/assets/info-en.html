MANUAL CUSTOM SETTINGS

There ir basic feature that XposedBridge API offer that can interfere Application value such as boolean, string, color, integer ,etc which called resparam.res.setReplacement(...) (for more info, please read Xposed wiki pages). with this MANUAL CUSTOM SETTINGS, you allow to modify string, boolean, color, integer and dimen. For now this feature only work for com.miui.home & com.android.systemui package .

  • boolean; 0=false 1=true
  • color; #FFFFFFFF
  • integer; any numbers format
  • dimen; numbers between 0-500
  • string; any chars
