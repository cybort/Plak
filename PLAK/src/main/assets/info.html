MANUAL CUSTOM SETTINGS

mengingat kebutuhan setiap manusia itu berbeda2, maka dengan itu, kita memanfaatkan fungsi dasar dari API xposed yaitu resparam.res.setReplacement(....) yang mana fungsi ini dapat dengan langsung mengintervensi value boolean, color, integer, dimen (jika kamu akrab sekali dengan grepe2 apk). MANUAL CUSTOM SETTINGS, untuk sementara dibatasi hanya untuk paket com.android.systemui & com.miui.home saja.

  • untuk value boolean, pada manual config settings, di isikan 1 jika nilainya true atau 0 jika nilainya false
  • untuk value color, diisikan dengan format i.e : #FFFFFFFF
  • untuk value integer, disikan dengan format angka
  • untuk value dimen, (mengingat dimen harus di anuin lewat anu) di isikan dengan angka antara 0-500
  • untuk value string, disikan dengan huruf

Oleh sebab fungsi custom ini sangat tak terbatas, maka akan sulit sekali untuk mengiringi value defaultnya. maka dari itu, untuk menonaktifkan kembali fitur ini, harus melakukan REBOOT terlebih dahulu setelah sebelumnya menonaktifkan menunya pada apps PLAK
